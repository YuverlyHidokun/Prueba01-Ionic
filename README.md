# Creación de una aplicación básica con Ionic

Creación de una aplicación usando las herramientas de Ionic para crear 4 tabs.

## Características

- Cálculo de raíces de ecuaciones cuadráticas.
- Interfaz intuitiva y fácil de usar.
- Compatible con múltiples plataformas gracias a Ionic.
- Almacenamiento local de datos.

## Tecnologías Utilizadas

- [Ionic](https://ionicframework.com/)
- [Angular](https://angular.io/)
- Capacitor para integración nativa.

## Instalación

Para instalar y ejecutar el proyecto, sigue estos pasos:

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu_usuario/nombre_del_repositorio.git

2. Accede al directorio del proyecto:
   ```bash
   cd nombre_del_repositorio

3. Instala las dependencias necesarias:
   ```bash
   npm install

3. Inicia la aplicación:
   ```bash
   ionic serve

Esto abrirá la aplicación en tu navegador en http://localhost:8100.
